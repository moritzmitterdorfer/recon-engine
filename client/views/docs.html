Install

NPM (recommended) soon...

For server side code, just install recon-engine and require the package in your node.js file. 

                        
const reconEngine = require('recon-engine-client');

CDN

For client side code (not recommended), you can also use our CDN. 

                        
<script src="http://reconengine.herokuapp.com/cdn/v1/recon.js"></script>

Constructor

Then, use the default constructor to connect your application to the recon cloud. Please insert your copied API key instead of YOUR_API_KEY. 

                        
// insert API KEY
let recon = new ReconClient({ token: 'YOUR_API_KEY' })

User actions

Whenever a user takes an action, send a message to our API using the SDK. 

                        
// userID and item are strings
recon.userAction(userID, item);

Train your model

When you have pushed enough data to our cloud, you can train a new model. This may take a few minutes. 

                        
// train the model (returns promise)
recon.trainModel()

Recommend items

When you have trained a model, you can start recommending items. You can do that by specifiying the userID. 

                        
// recommendations for a user
let recommendations = await recon.recommend(userID);

The response looks like following: 

                        
{
    message: "âœ… Recommendations for user ${userID} are given!",
    recommendations: [

        ...
    ]
}
