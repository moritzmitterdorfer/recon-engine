Install

NPM (recommended)

For server side code, just install recon-engine-client and require the package in your node.js file. 

                        
const ReconClient = require('recon-engine-client')

CDN

For client side code (not recommended), you can also use our CDN. 

                        
<script src="http://reconengine.herokuapp.com/cdn/v1/recon.js"></script>

Constructor

Then, use the default constructor to connect your application to the recon cloud. Please insert your copied API key instead of YOUR_API_KEY. In addition, you can pass in a calback which is called when the ReconClient is connected. 

                        
// insert API KEY
let recon = new ReconClient({ token: 'YOUR_API_KEY' })

// example
let recon = new ReconClient({ token: 'YOUR_API_KEY' }, () => {
    console.log('üç≠ Connected')
}));

User actions

Whenever an user takes an action, send a message to our API using the SDK. 

                        
// userID and item are strings
recon.userAction(userID, item)

Train your model

When you have pushed enough data to our cloud, you can train a new model. This may take a few minutes. Please take into consideration, that training should not be done after every user action! 

                        
// train the model (returns promise)
recon.trainModel()

// üî¥ only once or twice a day!

Recommend items

When you have trained a model, you can start recommending items. You can do that by specifying the userID. 

                        
// recommendations for an user
let recommendations = await recon.recommend(userID)

The response looks like following: 

                        
{
    message: "‚úÖ Recommendations for user ${userID} are given!",
    recommendations: [
        { item: 'itemID', confidence: 0.5 },
        { item: 'itemID', confidence: 0.5 },
        ...
    ]
}

Recommend items anonymously

You can also recommend items anonymously whithout creating an user. Just use an array of itemIDs containing items the user interacted with. In addition, you have to specify a valid callback. 

                        
recon.recommendAnonymously(itemIDArray, callback)

// example
recon.recommendAnonymously(['item_1', 'item_2'], recommendations => {
    console.log('üç≠ Recommendations', recommendations)
    // manipulate the DOM etc.
}))

Recommend items anonymously with storage (frontend only)

If you want to store the items the user has interacted with locally, you can use addItem. In addition, a new anonymous userID is created for that user. Whener you want to get recommendations for that user, call recommendAnonymouslyByStorage. 

                        
// add an item (works with cookies)
recon.addItem(itemID)

// get recommendations for local user 
recon.recommendAnonymouslyByStorage(callback)
